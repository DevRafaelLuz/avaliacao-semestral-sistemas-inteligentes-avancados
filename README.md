# Avaliação Semestral de Sistemas Inteligentes Avançados

Estes códigos implementam o treinamento e/ou classificação de três datasets diferentes.

## ⚙️ Ferramentas, Tecnologias e Bibliotecas

- Python
- Visual Studio Code
- pandas
- numpy
- scikit-learn
- imblearn
- pickle

## 🗃️ Datasets

> Heart Failure: [https://archive.ics.uci.edu/dataset/519/heart+failure+clinical+records](https://archive.ics.uci.edu/dataset/519/heart+failure+clinical+records)

> Wine Quality: [https://archive.ics.uci.edu/dataset/186/wine+quality](https://archive.ics.uci.edu/dataset/186/wine+quality)

> Black Friday Sales Dataset: [https://www.kaggle.com/datasets/noopurbhatt/retail-black-friday-sales-dataset](https://www.kaggle.com/datasets/noopurbhatt/retail-black-friday-sales-dataset)
